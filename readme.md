# Installation

For install any app, execute:

```bash
$ cd ~/Setup
$ ./install.sh
```

Then select one of the apps

# Setup

For setup any app, execute:

```bash
$ cd ~/Setup
$ ./setup.sh
```

Then select one of the apps

# Uninstall

For uninstall any app, execute:

```bash
$ cd ~/Setup
$ ./uninstall.sh
```

Then select one of the apps
